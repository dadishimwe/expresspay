# Express Payments Website

Complete payment processing website featuring Clover devices and transparent pricing.

## Features

- Responsive design with 14 rotating hero images
- Clover device showcases (Mini, Flex, Station Duo)
- Transparent pricing plans
- Client success stories
- Payment processing resources
- Contact pages

## Deployment

This site is deployed using GitHub Pages and can be accessed at:
- GitHub Pages: https://dadishimwe.github.io/expresspay/
- Custom Domain: https://newsite.domain.com

## Technology

- HTML5
- CSS3 (embedded)
- JavaScript (embedded)
- Webflow template (customized)
