## ⚙️ How to Replace Images

### Method 1: Direct File Replacement (Easiest)
1. Download/create your images
2. Upload to same CDN path
3. Images automatically update

### Method 2: Update HTML Links (More Control)
1. Create `/images/` folder in your site
2. Add your images to appropriate subfolders
3. Search and replace URLs in HTML:
   - Find: `https://cdn.prod.website-files.com/...`
   - Replace: `images/[category]/[filename]`

### Method 3: Use a CDN (Professional)
1. Upload images to a CDN (Cloudflare, AWS S3, etc.)
2. Update all src attributes with new CDN URLs
3. Better performance, scalability

## 🎨 Style Guidelines

### General Rules:
- **Consistency:** Use similar lighting, style, and tone across all photos
- **Quality:** High resolution (2x the display size for retina screens)
- **File Size:** Optimize images (use tools like TinyPNG or ImageOptim)
- **Format:** 
  - JPG for photos (80-90% quality)
  - PNG for transparency needs
  - SVG for logos and icons
  - WebP for modern browsers (with JPG fallback)

### Clover Device Photography:
- **Lighting:** Bright, professional, avoid harsh shadows
- **Background:** Clean (white/neutral) or real business environment
- **Angle:** Show screens clearly, demonstrate usability
- **Context:** Show devices in action when possible

### Color Palette (Current Site):
- Primary: `#0B2641` (dark blue)
- Accent: `#FF4006` (orange)
- Secondary: `#A37CFF` (purple)
- Background: `#FFFFFF` (white) / `#15151F` (dark)

Consider using these colors in your images or adjust the site's CSS to match your brand colors.

## 💡 Pro Tips

1. **Batch Process:** Use tools like Photoshop actions or online batch processors to resize multiple images at once

2. **Optimize Everything:** Run all images through compression before uploading
   - TinyPNG.com for PNG/JPG
   - SVGOMG for SVG files

3. **Use Placeholders:** If you don't have all images yet, use placeholder services:
   - placehold.co
   - via.placeholder.com
   - Just temporary until real images ready

4. **Test Responsive:** After replacing, test on mobile, tablet, and desktop to ensure images look good at all sizes

5. **Alt Text:** Update alt attributes to describe the actual content for SEO and accessibility
